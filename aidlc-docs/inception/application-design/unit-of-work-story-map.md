## 우선순위별 구현 순서

### Unit 1 (개발자 A) 구현 순서
1. US-A01, US-A02 (관리자 인증 — Unit 2에서도 사용)
2. US-C01 (테이블 자동 로그인)
3. US-C02~C04 (메뉴 조회)
4. US-C05~C10 (장바구니)
5. US-C11~C12 (주문 생성)
6. US-C13~C16 (주문 내역/에러 처리)
7. US-A03 (로그인 시도 제한)

### Unit 2 (개발자 B) 구현 순서
1. US-A09 (테이블 초기 설정)
2. US-A14 (메뉴 등록 — 테스트 데이터 필요)
3. US-A04~A05 (대시보드 + SSE)
4. US-A06~A07 (주문 상세/상태 변경)
5. US-A10~A11 (주문 삭제/이용 완료)
6. US-A12 (과거 내역)
7. US-A08, A13, A15~A17 (필터링, 메뉴 관리 부가기능)
