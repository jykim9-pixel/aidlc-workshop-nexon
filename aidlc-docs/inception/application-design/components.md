# 컴포넌트 정의 (Components)

## 백엔드 컴포넌트 (packages/backend)

### 1. Auth Module
**책임**: 인증 및 세션 관리
- 관리자 로그인/로그아웃 처리
- JWT 토큰 발급 및 검증
- 테이블 태블릿 인증
- 로그인 시도 제한

### 2. Store Module
**책임**: 매장 정보 관리
- 매장 기본 정보 조회

### 3. Table Module
**책임**: 테이블 및 세션 관리
- 테이블 CRUD
- 테이블 세션 라이프사이클 (시작/종료)
- 테이블 초기 설정 (비밀번호)

### 4. Menu Module
**책임**: 메뉴 및 카테고리 관리
- 카테고리 CRUD
- 메뉴 항목 CRUD
- 메뉴 노출 순서 관리

### 5. Order Module
**책임**: 주문 처리 및 관리
- 주문 생성
- 주문 상태 변경
- 주문 삭제
- 주문 조회 (현재 세션 / 과거 이력)

### 6. SSE Module
**책임**: 실시간 이벤트 스트리밍
- SSE 연결 관리
- 주문 이벤트 브로드캐스트
- 연결 상태 모니터링

### 7. Middleware
**책임**: 공통 미들웨어
- JWT 인증 미들웨어
- 에러 핸들링 미들웨어
- 요청 유효성 검증 미들웨어
- CORS 설정

## 프론트엔드 컴포넌트 - 고객용 (packages/customer-app)

### 1. Auth (자동 로그인)
**책임**: 태블릿 인증 관리
- 저장된 인증 정보로 자동 로그인
- 초기 설정 화면 (최초 1회)
- 인증 상태 관리

### 2. Menu (메뉴 조회)
**책임**: 메뉴 탐색 UI
- 카테고리 네비게이션
- 메뉴 카드 그리드
- 메뉴 상세 모달

### 3. Cart (장바구니)
**책임**: 장바구니 상태 및 UI
- 장바구니 항목 표시
- 수량 조절 UI
- 총 금액 계산/표시
- 로컬 스토리지 persist

### 4. Order (주문)
**책임**: 주문 생성 및 내역 조회
- 주문 확인 화면
- 주문 성공/실패 화면
- 주문 내역 목록

### 5. Common UI
**책임**: 공통 UI 컴포넌트
- 버튼, 모달, 토스트
- 로딩 스피너
- 에러 바운더리

## 프론트엔드 컴포넌트 - 관리자용 (packages/admin-app)

### 1. Auth (관리자 인증)
**책임**: 관리자 로그인/세션 관리
- 로그인 폼
- JWT 토큰 관리
- 자동 로그아웃 (16시간)

### 2. Dashboard (주문 모니터링)
**책임**: 실시간 주문 대시보드
- 테이블별 카드 그리드
- SSE 연결 및 실시간 업데이트
- 주문 상세 모달
- 신규 주문 강조 표시

### 3. Table Management (테이블 관리)
**책임**: 테이블 운영 관리
- 테이블 초기 설정
- 주문 삭제
- 이용 완료 처리
- 과거 내역 조회

### 4. Menu Management (메뉴 관리)
**책임**: 메뉴 CRUD UI
- 메뉴 목록/등록/수정/삭제
- 카테고리 관리
- 순서 조정 (드래그 앤 드롭)

### 5. Common UI
**책임**: 공통 UI 컴포넌트
- 버튼, 모달, 확인 팝업
- 데이터 테이블
- 필터/검색

## 공유 패키지 (packages/shared)

### 1. Types
**책임**: 공유 TypeScript 타입 정의
- API 요청/응답 타입
- 엔티티 타입 (Order, MenuItem, Table 등)
- Enum (OrderStatus, etc.)

### 2. Constants
**책임**: 공유 상수
- 주문 상태 값
- API 경로 상수
- 에러 코드

### 3. Utils
**책임**: 공유 유틸리티 함수
- 금액 포맷팅
- 날짜 포맷팅
- 유효성 검증 헬퍼
